## Run Locally 

Clone the project

```bash
  git clone git@github.com:ankitbisen28/Food-order-app.git
```

Go client Folder 
```   
  cd client
```

Install Fontend dependencies

```bash
  npm install
```

## Open another Terminal 
Open another Terminal For Backend 

Go Backend Folder 
```   
  cd backend
```

Install backend dependencies

```bash
  npm install
```
## Open Root Terminal
Start your server on root

```bash
  npm start
```
